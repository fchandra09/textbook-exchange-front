# Textbook Exchange (Front-End)

*University of Illinois at Urbana-Champaign  
The Art of Web Programming (CS498RK) Fall 2016 Final Project*  

## Development Instructions
1. `npm install`
2. `bower install`
3. `grunt` or `grunt serve`

## Production Instructions
1. `npm install`
2. `bower install`
3. `grunt build`
4. `node app.js`
